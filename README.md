# Task 01: Data Visualization

A simple Python project to visualize the distribution of categorical and continuous variables using a bar chart and a histogram. This project fulfills **Task-01** of the Prodigy InfoTech Data Science internship.

---

## 📊 Features
* **Bar Chart:** Visualizes the distribution of a categorical variable (`Sex`).
* **Histogram:** Visualizes the distribution of a continuous variable (`Age`).

## 📂 Dataset
The project utilizes the classic **Titanic Dataset** fetched directly from the live web URL:
`https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv`
